Operator perbandingan

== : digunakan untuk membandingkan 2 buah nilai apakah sama atau tidak tanpa membandingkan tipe datanya (5 == 5)
!= : digunakan untuk membandingkan 2 buah nilai apakah berbeda atau tidak tanpa membandingkan tipe datanya (5 != 4)
=== : digunakan untuk membandingkan 2 buah nilai apakah sama atau tidak dan membandingkan tipe datanya (5 === 5)
!== : digunakan untuk membandingkan 2 buah nilai apakah berbeda atau tidak dan membandingkan tipe datanya (5 !== 4)
> dan < : digunakan untuk membandingkan 2 buah nilai apakah salah satunya lebih besar atau lebih kecil dari nilai satunya lagi (1 < 3 atau 3 > 1)
>= dan <= : digunakan untuk membandingkan 2 buah nilai apakah salah satunya lebih besar atau sama dengan dan juga lebih kecil atau sama dengan dari nilai satunya lagi (1 <= 3 atau 3 >= 1)
! : digunakan untuk mengubah sebuah nilai menjadi negatif (kebalikannya) (true -> false)
